# Reversing the equation
## Easy
<div class="problems_problem_content__Xm_eO"><p><span style="font-size:18px">Given a mathematical equation using numbers/variables and +, -, *, /. Print the equation in reverse.</span></p>

<p><span style="font-size:18px"><strong>Example 1:</strong></span></p>

<pre><span style="font-size:18px"><strong>Input:</strong>
S = "20-3+5*2"
<strong>Output:</strong> 2*5+3-20
<strong>Explanation</strong>: The equation is reversed with
numbers remaining the same.</span>
</pre>

<p><span style="font-size:18px">â€‹<strong>Example 2:</strong></span></p>

<pre><span style="font-size:18px"><strong>Input</strong>: 
S = "a+b*c-d/e"
<strong>Output:</strong> e/d-c*b+a
<strong>Explanation</strong>: The equation is reversed with
variables remaining the same.</span>
</pre>

<p><br>
<span style="font-size:18px"><strong>Your Task:</strong><br>
You don't need to read input or print anything. Your task is to complete the function&nbsp;<strong>reverseEqn()&nbsp;</strong>which takes the string S representing the equaltion as input and returns the resultant string representing the equaltion in reverse.</span></p>

<p><br>
<span style="font-size:18px"><strong>Expected Time Complexity:&nbsp;</strong>O(|S|).<br>
<strong>Expected Auxiliary Space:&nbsp;</strong>O(|S|).</span></p>
